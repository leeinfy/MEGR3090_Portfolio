Blynk Assignment


Material Used: Raspberry Pi, LED, Camera module, DC motor, L298N motor drive,Blynk.

The wiring is almost the same as My python setup. The raspberrypi is connect with a motordrive. It is also connected with LED. The Blynk project is use the Blynk phone app to connect the motor and LED. The LED is connect to pin4 and a 330ohm resistor. the motor drive motor1 pin 1 , pin2 and speed pin are connect to GPIO 17, 27, and 22. The camera module is also mounted on the raspberry pi.


Motor python prototype Code

This program cycle the mode everytime the button is pressed. the first mode is servo in its 90 degree postion, and the dc motor is stopped. The second mode is servo is 0 degree position, and dc motor drive forward. the third mode is servo is 180 degree position, and the dc motor in backward mode. I was original want to use the joystick to control the speed of motor. Since the joystick need extra circuit for analog input. I only wrote the 50 PWM dutycycle. And in the buttonispressed function is for button debouncing issue.
